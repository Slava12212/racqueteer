## ✏️ Як редагувати сторінку

1. Зайдіть у **WP Admin → Pages**
2. Натисніть на потрібну сторінку
3. У редакторі **натисніть на блок**, який хочете змінити
4. Поля блоку з'являться у **правій панелі**
5. Відредагуйте потрібні поля
6. Натисніть **Update** (верхній правий кут)
7. Сайт оновиться автоматично за **1–5 секунд**

> ⏳ Якщо зміни не з'явились одразу — зачекайте до 60 секунд. Якщо ще не оновилось — зверніться до розробника.

## 📦 Управління записами

Записи — це окремі одиниці контенту (вакансії, відгуки, тарифи тощо), які автоматично відображаються у відповідних секціях сайту.

---

### Вакансії (Jobs)

**Де:** `WP Admin → Jobs`

| Поле | Опис |
|------|------|
| **Title** | Назва вакансії (напр. "Head Padel Coach") |
| **Category** | Категорія ролі (напр. "Coaching", "Operations") |
| **Description** | Короткий опис позиції |

**Додати вакансію:**
1. `WP Admin → Jobs → Add New`
2. Заповніть Title, Category та Description
3. Натисніть **Publish** — вакансія з'явиться на сайті за кілька секунд

**Приховати вакансію без видалення:**
- Відкрийте вакансію → змініть статус на **Draft** → натисніть **Update**
- Вакансія зникне з сайту одразу ж

---

## 🖼️ Робота з медіафайлами

### Завантаження зображень
`WP Admin → Media → Add New` — перетягніть файл або натисніть для вибору.

### Використання зображень у полях
Поля типу **Image** та **Gallery** відкривають медіатеку. Ви можете:
- вибрати вже завантажений файл
- завантажити новий прямо з поля

### Відеофайли
Для відео потрібно вписати шлях до файлу у текстове поле (напр. `/hero-video-opt.mp4`).
Щоб додати нове відео — зверніться до розробника, він завантажить файл на сервер.

## ➕ Створення нових сторінок

1. `WP Admin → Pages → Add New`
2. Задайте назву та **slug** (адресу, напр. `summer-camp`)
3. Додайте блоки через кнопку `+` → категорія **Racqueteer**
4. Заповніть поля блоків
5. Натисніть **Publish**
6. Сторінка з'явиться на сайті автоматично за адресою `https://racqueteer.com/summer-camp`

**Приховати сторінку без видалення:**
- Змініть статус на **Draft** → сторінка стає недоступною для відвідувачів
- Поверніть статус **Published** → знову стає доступною

## 🐛 Що робити, якщо щось не працює

### Зміни не з'явились після збереження

1. Ви точно натиснули **Update / Publish** (не просто закрили вкладку)?
2. Зачекайте до 60 секунд — сайт оновлюється автоматично
3. Спробуйте відкрити сторінку в режимі **інкогніто** браузера (без кешу)
4. Якщо все ще не змінилось — зверніться до розробника

---

### Вакансія або відгук не відображається на сайті

- Перевірте, що запис має статус **Published** (не Draft)
- Перевірте, що всі обов'язкові поля заповнені (Category, Description)

---

### Нова сторінка не відкривається (помилка 404)

- Переконайтесь, що статус сторінки — **Published**
- Переконайтесь, що на сторінці є хоча б один блок

---

### Логотип у шапці або підвалі показує не те зображення

- `WP Admin → Site Settings → Navbar` → перевірте поле **Logo**
- `WP Admin → Site Settings → Footer` → перевірте поле **Footer Logo**
- Якщо поле порожнє — завантажте або виберіть зображення

---

### Усі картки програм показуються синім кольором

- Відкрийте кожну програму у `WP Admin → Programs`
- Переконайтесь, що поле **Color** встановлено у `Red` або `Blue`
- Збережіть запис (**Update**)
- Якщо після збереження колір не змінився — зверніться до розробника
